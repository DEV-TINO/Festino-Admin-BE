# Festino-Admin-BE
Festino Website Backend Project by using `Spring Boot`

# Skills
- Spring Boot

# Installation

## Using docker
### Go to festino-admin directory
```bash
cd ./festino-admin
```
### Build docker image
```bash
docker compose build
```
### Run in background
```bash
docker compose up
```

### Show logs
```bash
docker compose logs -f
```
